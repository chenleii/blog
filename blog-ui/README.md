# 前端ui

使用 [Ant Design Pro](https://pro.ant.design) 进行搭建。更多内容请自行查看完整文档。

## 环境准备

```bash
# 安装node省略......

# 安装依赖
npm i

# 启动项目，连接本地后端api。
npm run start:dev

# 打包
npm run build
```
